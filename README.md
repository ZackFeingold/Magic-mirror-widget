

# MagicMirror 
This project was created as a custom extension for a Raspberry Pi based smart mirror setup.

# Hebrew Word of the Day Widget

A custom module for MagicMirror that displays a Hebrew word of the day.


## Project Overview

Built a smart mirror using a Raspberry Pi, two-way mirror glass and a repurposed monitor to display real-time information such as time, weather and calendar updates. Independently researched hardware and software requirements through online forums and open-source documentation, assembling and configuring the system from scratch.

The aim was to extend the MagicMirror platform with a simple custom information module and integrate it into the existing dashboard.

## What I Built

I:
- configured the Raspberry Pi MagicMirror environment
- built a custom module for Hebrew word display
- customised the module styling with CSS
- integrated the widget into the mirror layout and configuration

## System Context

The wider smart mirror setup used:
- Raspberry Pi
- repurposed monitor
- two-way mirror glass
- MagicMirror software platform

This repository focuses specifically on the custom Hebrew word widget rather than the full hardware build.

## Installation

1. Copy the module into the MagicMirror modules directory
2. Add the config entry to config.js
3. Restart MagicMirror
